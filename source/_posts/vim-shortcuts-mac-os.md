### Basics

|   Keys   |    Command   |
|----------|:-------------:|
| $ (Shift+4) | To the end of the line |
| ^ (Shift+6) / 0 | To the start of the line |


### Delete

|   Keys   |    Command   |
| dd | delete the entire line |
| dw | delete the word |
| d$ | delete from the current cursor to the end of the line |
